# Content Management

Content Management covers topics that are important to keeping your Cascade CMS website up to date and getting the most out of your Cascade CMS installation.

## [Reports](https://www.hannonhill.com/cascadecms/latest/content-management/reports/index.html)

Cascade CMS provides a number of reports geared towards monitoring user activity and content creation and management.

## [Active Users Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/active-users-report.html)

The Active Users report provides a list of users logged into the system.

## [User Activity Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/user-activity-report.html)

## [Background Tasks](https://www.hannonhill.com/cascadecms/latest/content-management/reports/background-tasks.html)

The Background Tasks report provides a list of completed and running tasks in the system.

## [Content Inventory Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/content-inventory-report.html)

The Content Inventory Report provides users with a comprehensive overview of their site's assets, allowing for filtering by various criteria such as asset types, publish status, and owners. Users can assign content ownership, identify stale content, and send notifications for assets that require review. Additionally, users can schedule review dates and export report results as a CSV file for further analysis.

## [Stale Content Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/stale-content-report.html)

The Stale Content report displays a list of assets that have not been modified within a specified time period.

## [Content Ownership Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/content-ownership-report.html)

The Content Ownership report displays assets owned by users in the system.

## [Broken Link Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/broken-links-report.html)

The Broken Links Report provides the results of the scheduled broken Link Checker which can be configured in your System Preferences.

## [Workflows Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/workflows-report.html)

The Workflows report displays a list of in-progress and/or completed workflows within sites that you can access.

## [Content Reviews Report](https://www.hannonhill.com/cascadecms/latest/content-management/reports/content-up-for-review-report.html)

The Content Reviews report provides a list of assets that have upcoming or past-due review dates.

## [Workflows](https://www.hannonhill.com/cascadecms/latest/content-management/workflows/index.html)

A workflow is a series of steps that content must go through before it can be created, edited, copied, moved, renamed, and/or deleted by users in Cascade CMS.

## [Custom Workflow Emails](https://www.hannonhill.com/cascadecms/latest/content-management/workflows/custom-workflow-emails.html)

Custom Workflow Emails allow you to personalize your workflow notification and completion emails. A WYSIWYG editor plus dropdowns to insert placeholders for workflow details makes it simple to build templates for informative and actionable emails.

## [Workflow Triggers](https://www.hannonhill.com/cascadecms/latest/content-management/workflows/workflow-triggers/index.html)

A workflow trigger is a plugin that enhances an action in the workflow process by executing code as the transition from a source step to a destination step occurs. It encapsulates some system logic to accomplish a non-workflow related function.

## [Assigning Workflows](https://www.hannonhill.com/cascadecms/latest/content-management/workflows/how-do-i-assign-a-workflow.html)

In Cascade CMS, workflows can either be used as part of an Asset Factory for the creation of new content, or assigned to a folder to control actions taken on contained content.

## [Receiving Workflow Notifications](https://www.hannonhill.com/cascadecms/latest/content-management/workflows/receiving-workflow-notifications.html)

You can be notified of waiting workflows by email or in the My Workflows dashboard widget.

## [Workflow XML Schema Reference](https://www.hannonhill.com/cascadecms/latest/content-management/workflows/workflow-xml-schema-reference.html)

## [Content Review](https://www.hannonhill.com/cascadecms/latest/content-management/content-review.html)

The Review Date metadata field helps you to keep your content up to date by allowing you to schedule reviews for assets.

## [Tasks](https://www.hannonhill.com/cascadecms/latest/content-management/tasks.html)

Tasks allow content managers and contributors to create and organize their to-do lists in Cascade CMS.
