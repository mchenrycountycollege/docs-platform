## Purpose

This doc exists so brand, navigation, and footer alignment across our digital
properties is a repeatable, documented process — not a case-by-case judgment
call every time the main site gets redesigned.

## How to use this template

For each property in your portfolio: answer the two questions in the tiering
rubric below, assign a tier, fill in a row in the properties inventory, and
complete the matching handoff kit checklist. Re-run the audit cadence section
annually.

## Tiering rubric

Assess each property against two questions:

1. **Technical control** — can this property ingest a shared, literal
   component (include/API/synced template), or is it configured through a
   vendor's limited theme editor?
2. **Brand risk / audience overlap** — is this property likely to be seen by
   prospective students, parents, or donors, or is it low-visibility/internal?
